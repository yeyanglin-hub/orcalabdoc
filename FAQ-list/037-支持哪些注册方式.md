# 支持哪些注册方式？

## 问题
OrcaLab账户注册除了传统的邮箱/手机注册外，还支持哪些便捷的第三方注册方式？

## 回答

OrcaLab为了方便用户，提供了多种账户注册方式，包括**手机注册、邮箱注册，以及第三方账号注册（微信和GitHub）**。

## 📋 OrcaLab支持的注册方式

### 1. **手机注册**
- **特点**：通过国内手机号接收短信验证码进行注册。
- **优势**：便捷、安全，尤其适合中国大陆用户。
- **流程**：
  1. 在注册界面选择手机注册。
  2. 输入手机号，获取并输入短信验证码。
  3. 填写个人信息（名、姓、用户名、密码等）。
  4. 接受用户协议并提交注册。

### 2. **邮箱注册**
- **特点**：通过邮箱地址接收验证码进行注册。
- **优势**：国际通用，适用于没有国内手机号或偏好邮箱管理账户的用户。
- **流程**：
  1. 在注册界面选择邮箱注册。
  2. 输入邮箱地址，获取并输入邮箱验证码。
  3. 填写个人信息（名、姓、用户名、密码等）。
  4. 接受用户协议并提交注册。

#### ⚠️ 邮箱注册注意事项
- **Gmail邮箱问题**：由于Google的安全策略，Gmail邮箱可能会出现收不到验证码的情况。建议优先使用其他邮箱（如QQ邮箱、163邮箱等）。

### 3. **第三方账号注册**

OrcaLab目前支持通过以下两种主流第三方平台进行快捷注册和登录。

#### 🚀 微信注册
- **特点**：通过微信扫码授权，实现快速注册和登录。
- **优势**：对于微信用户来说非常方便，省去了填写繁琐信息的步骤。
- **流程**：
  1. 在注册界面点击微信图标。
  2. 使用微信扫描网页上显示的二维码进行授权。
  3. 授权成功后，系统会引导您绑定一个邮箱地址（用于找回密码和接收通知）。
  4. 填写少量必要个人信息（名、姓、用户名、手机号、密码等）。
  5. 接受用户协议并提交注册。

#### 💻 GitHub注册
- **特点**：通过GitHub账号授权，实现注册和登录，尤其适合开发者和科研人员。
- **优势**：方便管理，与开发工作流更紧密结合。
- **流程**：
  1. 在注册界面点击GitHub图标。
  2. 输入您的GitHub账号邮箱，并根据GitHub的授权页面提示进行操作。
  3. 授权成功后，系统会引导您绑定一个邮箱地址。
  4. 填写少量必要个人信息（名、姓、用户名、手机号、密码等）。
  5. 接受用户协议并提交注册。

## 📝 注册成功后的体验

无论采用哪种注册方式，注册成功后系统都会自动登录您的OrcaLab账户，并进入资产库界面。您将获得：
- 访问和订阅OrcaLab资产库的权限。
- 使用AI生成3D资产工具的额度。
- 享受OrcaLab提供的各项非商用服务。


## 💡 注册建议

- **手机/邮箱注册**：如果您希望完全独立管理账户，或者不希望与第三方平台绑定，推荐使用。
- **微信/GitHub注册**：如果您日常习惯使用这些平台，可以快速注册，但后续可能需要绑定一个备用邮箱。
- **牢记密码**：无论哪种方式，请妥善保管您的密码，并选择一个强密码。

选择最适合您的注册方式，即可开始您的OrcaLab之旅！
