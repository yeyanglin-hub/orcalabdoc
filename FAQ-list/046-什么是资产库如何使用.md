# 什么是资产库？如何使用？

## 问题
OrcaLab中的资产库是什么？包含哪些内容？如何有效使用资产库进行仿真项目开发？

## 回答

**资产库**是OrcaLab的核心组件之一，提供了丰富的3D模型、场景、机器人等数字资产，是构建仿真环境的重要资源平台。

## 🏪 资产库概述

### 定义与作用
资产库是一个**集中化的数字资产管理平台**，为用户提供：
- 3D模型和场景资产
- 机器人模型与配置
- 物理仿真组件
- AI训练环境素材


### 访问方式
```bash
# 方法1：从OrcaLab客户端
点击【打开资产库】按钮

# 方法2：直接访问网址
https://simassets.orca3d.cn/
```

## 📁 资产库分类结构

### 🏭 6大主要分类

#### 1. **工业资产**
```
内容：机械设备、生产线、工业机器人
应用：制造业仿真、工业自动化培训
示例：机械臂、传送带、装配线
```

#### 2. **生活资产**  
```
内容：家具、家电、日用品
应用：服务机器人训练、生活场景仿真
示例：桌椅、冰箱、餐具
```

#### 3. **服务资产**
```
内容：服务场所、设施设备
应用：服务行业仿真、场景建模
示例：餐厅、医院、商店
```

#### 4. **传感器资产**
```
内容：各类传感器模型
应用：感知系统仿真、数据采集
示例：相机、激光雷达、IMU
```

#### 5. **机器人资产**
```
内容：各类机器人模型与配置
应用：机器人控制、行为训练
示例：四足机器人、机械臂、移动底盘
```

#### 6. **其他资产**
```
内容：场景道具、特殊用途资产
应用：辅助建模、特定需求
示例：地形、植物、装饰物
```



## 🔍 资产搜索功能

### 搜索方式

#### 🖼️ 图片搜索
```
功能：通过上传参考图片搜索相似资产
适用：有具体物体图片时快速查找
操作：上传图片 → AI识别 → 匹配结果
```

#### 📝 文字描述搜索
```
功能：通过关键词文字描述搜索
适用：知道大概需求但没有具体图片
操作：输入描述 → 关键词匹配 → 展示结果
```


### 搜索技巧
```
关键词策略：
├── 使用物体名称 (如"桌子"、"机械臂")
├── 使用功能描述 (如"抓取"、"移动")  
├── 使用场景需求 (如"厨房"、"工厂")
└── 使用品牌型号 (如"UR5"、"Panda")

组合搜索：
├── "工业 机械臂 抓取"
├── "移动机器人 导航 激光"
└── "四足 行走 动态"
```

## 📦 资产包管理

### 资产包概念
- **资产包**是资产库的基本管理单位
- 一个资产包包含1个或多个相关资产
- 以包为单位进行订阅和下载

### 资产包信息
```
基本信息：
├── 作者信息
├── 项目名称  
├── 资产数量
├── 总文件大小
├── 版本信息
└── 更新时间

详细信息：
├── 资产描述
├── 使用说明
├── 版本更新日志
├── 依赖关系
└── 兼容性信息
```



## 🔄 资产订阅流程

### 订阅步骤
```
1. 浏览资产库 → 查找目标资产包
2. 查看详情 → 确认资产内容和要求
3. 点击订阅 → 添加到个人资产库
4. 重启OrcaLab → 自动下载同步
5. 资产可用 → 开始使用资产
```

### 订阅管理
```bash
订阅后资产包会：
├── 出现在个人中心
├── 自动下载到本地
├── 在OrcaLab客户端可用  
└── 支持版本更新

取消订阅：
├── 从个人中心移除
├── 本地文件保留
├── 客户端中不可用
└── 可重新订阅
```



## 🤖 AI生成工具

### 工具类型

#### 🎨 文生3D
```
功能：根据文字描述生成3D模型
输入：文字描述 (如"红色的圆形桌子")
输出：USDZ格式3D资产
限额：每天5次
```

#### 🖼️ 图生3D  
```
功能：根据参考图片生成3D模型
输入：2D图片文件
输出：USDZ格式3D资产  
限额：每天5次
```


### 生成参数说明
```
参数配置（建议保持默认）：
├── 分离Mesh：拆分模型组件
├── 自动生成LOD：多层次细节
├── 平滑Mesh：表面平滑处理
├── 平滑强度：处理程度控制
└── 使用Hub CLI：处理工具选择
```

### AI生成最佳实践
```
文字描述技巧：
├── 描述具体 ("红色木质圆桌"比"桌子"更好)
├── 包含材质 ("金属"、"塑料"、"木质")
├── 指定尺寸 ("小型"、"中等"、"大型") 
└── 添加细节 ("带抽屉"、"四条腿")

图片选择技巧：  
├── 清晰度高、光线充足
├── 目标物体居中、完整
├── 背景简洁、干扰少
└── 角度展示物体主要特征
```

## 📱 资产库使用工作流

### 🎯 项目规划阶段
```
1. 明确仿真需求
   ├── 场景类型 (室内/室外/工业)
   ├── 机器人类型 (移动/机械臂/四足)
   ├── 任务目标 (抓取/导航/检测)
   └── 精度要求 (教学/研究/产品)

2. 搜索相关资产
   ├── 按分类浏览
   ├── 关键词搜索  
   ├── 参考图片搜索
   └── 记录目标资产包
```

### 🔧 资产准备阶段
```
3. 订阅必要资产
   ├── 核心场景资产
   ├── 主要机器人模型
   ├── 必需的传感器
   └── 辅助道具资产

4. 资产下载同步
   ├── 重启OrcaLab触发下载
   ├── 等待同步完成
   ├── 检查资产完整性
   └── 验证资产可用性
```

### 🚀 项目开发阶段
```
5. 构建仿真场景
   ├── 选择基础场景
   ├── 拖拽添加资产
   ├── 调整位置姿态
   └── 配置物理属性

6. 测试与优化
   ├── 运行基础仿真
   ├── 检查资产行为
   ├── 优化性能表现
   └── 保存场景布局
```

## ⚠️ 使用注意事项

### 版权与使用限制
- 资产库资产仅限非商用用途
- 不得二次分发或商业化使用
- AI生成资产同样受限于非商用条款
- 尊重原作者版权声明

### 性能考虑
- 大型资产包会占用较多存储空间
- 复杂资产会影响仿真性能
- 建议根据硬件配置合理选择资产
- 定期清理不需要的资产包

资产库是OrcaLab生态系统的重要组成部分，合理利用资产库资源可以大大提高仿真项目的开发效率和质量。
