# 订阅资产后如何在OrcaLab中使用？

## 问题
我在OrcaLab资产库中成功订阅了3D资产包。我应该如何在OrcaLab客户端中找到这些资产，并将它们添加到我的仿真场景中使用？

## 回答

当您在OrcaLab资产库中订阅资产包后，OrcaLab客户端会自动同步这些资产到您的本地。您可以在客户端的"资产栏"中找到它们，并拖拽到仿真场景中使用。

## 📋 在OrcaLab客户端中使用订阅资产流程

### 第一步：确保资产已同步到本地

1.  **重启OrcaLab客户端**：在资产库网页端订阅资产后，必须**关闭并重新启动OrcaLab客户端**，才能触发资产的下载和同步。
    ```bash
    # 激活 OrcaLab 的 Conda 环境
    conda activate orcalab
    # 启动 OrcaLab
    orcalab
    ```
2.  **观察同步过程**：客户端启动时会显示"正在同步资产包..."的提示。请等待此过程完成，确保资产已成功下载。



### 第二步：打开OrcaLab客户端界面

成功启动并同步资产后，您会进入OrcaLab的GUI界面。



### 第三步：导航到"资产栏"

-   在OrcaLab客户端界面的底部（或根据您的布局设置），找到并点击"资产"（Assets）标签页，进入"资产栏"模块。



### 第四步：搜索或浏览所需资产

-   **搜索功能**：在资产栏顶部的搜索框中输入资产名称的关键词，可以快速过滤和查找已下载的资产。
-   **浏览功能**：资产栏会以列表或网格形式展示您已订阅并下载的所有资产。您可以滚动查看。

### 第五步：将资产拖拽到视图中

-   **拖拽操作**：找到您想要使用的资产后，用鼠标点击该资产，将其**拖拽到中心"视图"区域**（3D场景中）。
-   **放置**：在视图中释放鼠标，资产就会出现在场景中。

### 第六步：调整资产的Transform属性

-   **选中资产**：在"视图"中点击您刚刚放置的资产，或在"大纲栏"中选择该资产。
-   **编辑属性**：在右侧的"编辑栏"中，您可以调整资产的：
    -   `Position` (位置)：X、Y、Z坐标
    -   `Rotation` (旋转)：X、Y、Z轴旋转角度
    -   `Uniform Scale` (统一缩放)：整体缩放比例



### 第七步：保存场景布局（可选）

-   如果您对场景的搭建和资产的摆放满意，建议通过"文件"菜单，选择"保存布局"或"另存为"，将当前场景配置保存为一个布局文件（通常是`.json`或`.orcalab`文件）。
-   这样，下次您可以直接加载这个布局文件，而无需重新搭建场景。

## 💡 使用注意事项

### 1. **订阅状态与本地文件**
- 只有在资产库中**成功订阅**的资产，OrcaLab客户端才会尝试同步下载。
- 即使本地有资产文件，如果未订阅，客户端也可能不会在资产栏中显示。

### 2. **资产包类型**
- 有些资产包可能带有`有场景`标签，意味着它们自带一个预设的场景布局。订阅这类资产后，在OrcaLab启动时的"选择场景"对话框中，您就可以直接选择该资产包提供的场景。

### 3. **网络和磁盘空间**
- 确保在资产同步时有良好的网络连接。
- 确保本地有足够的磁盘空间来存储下载的资产包。

### 4. **资产搜索的精确性**
- 在资产栏中搜索时，如果资产名称较长或有特殊字符，请尝试使用部分关键词。

## 📝 总结

OrcaLab提供了一个无缝的资产使用流程：在资产库网页端订阅，重启客户端自动同步，然后在客户端"资产栏"中拖拽使用，并通过"编辑栏"进行精细调整。掌握这个流程是高效构建仿真场景的基础。
