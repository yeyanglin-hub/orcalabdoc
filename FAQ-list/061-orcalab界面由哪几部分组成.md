# OrcaLab界面由哪几部分组成？

## 问题
OrcaLab的用户界面是如何布局的？各个模块分别有什么功能？

## 回答

OrcaLab采用模块化的用户界面设计，支持灵活布局。默认界面分为**上、中、下三层，共五个核心模块**，为用户提供完整的仿真开发环境。

## 🖥️ 整体界面布局



### 界面结构概览
```
┌─────────────────────────────────────────┐
│                菜单栏                    │  ← 顶层
├─────────────┬─────────────┬─────────────┤
│   大纲栏    │   视图栏     │   编辑栏    │  ← 中层  
├─────────────┴─────────────┴─────────────┤
│                资产栏                    │  ← 底层
└─────────────────────────────────────────┘
```

## 📋 菜单栏模块

### 🔧 顶部信息区
```
左侧控制按钮：
├── 隐藏按钮 [-]     - 最小化窗口
├── 最大化按钮 [□]   - 全屏/窗口切换  
└── 关闭按钮 [×]     - 退出程序

中部信息显示：
└── 版本号显示 [26.1.2] - 当前OrcaLab版本
```



### 🛠️ 功能菜单区
```
左侧菜单：
├── 文件 (File)
│   ├── 打开布局 (Ctrl+O)
│   ├── 保存布局 (Ctrl+S) 
│   ├── 另存为 (Ctrl+Shift+S)
│   ├── 新建布局 (Ctrl+N)
│   └── 退出
├── 编辑 (Edit)  
│   ├── 撤销 (Ctrl+Z)
│   └── 重做 (Ctrl+Shift+Z)
└── 帮助 (Help)
    └── 关于OrcaLab
```

### ⚙️ 操作工具区
```
中部工具按钮：
├── 移动工具 [↔] (快捷键: 1)  - 移动选中物体
├── 旋转工具 [↻] (快捷键: 2)  - 旋转选中物体  
└── 缩放工具 [⤢] (快捷键: 3)  - 缩放选中物体

右侧仿真控制：
├── 仿真运行 [▶] - 开始仿真
└── 停止仿真 [⏹] - 结束仿真
```

## 🗂️ 大纲与相机栏

### 功能切换
```
模式切换：
├── 大纲模式 - 管理场景物体层级结构
└── 相机模式 - 管理和切换视图相机
```



### 🌳 大纲模式功能
```
资产管理功能：
├── 显示场景中所有物体的层级结构
├── 支持群组创建和管理
├── 资产重命名和删除
├── 拖拽调整层级关系
└── 快速选择和定位物体

操作方式：
├── 右键菜单 - 添加群组、删除、重命名
├── 左键选择 - 选中物体（同步到视图）
├── 拖拽操作 - 调整父子关系
└── Delete键 - 快速删除选中物体
```

### 📷 相机模式功能
```
相机管理：
├── 列出所有可用相机
├── 切换不同相机视角
├── 添加自定义相机
└── 管理相机参数

默认相机：
├── 主视图相机 - 默认3D视角
├── 正交相机 - 正视图、侧视图等
└── 自由相机 - 用户自定义视角
```



## 👁️ 视图栏（核心区域）

### 3D视图功能
```
主要功能：
├── 3D场景实时渲染和显示
├── 物体选择和视觉反馈  
├── 实时仿真结果展示
├── 交互式场景编辑
└── 多种渲染模式支持
```



### 视图控制操作
```
鼠标操作：
├── 左键点击 - 选择物体
├── 右键拖拽 - 旋转视角
├── 滚轮滚动 - 缩放视图
└── 中键拖拽 - 平移视图

键盘操作：
├── 右键+W/S - 前进/后退
├── 右键+A/D - 左移/右移  
├── 右键+Q/E - 上移/下移
├── 右键+Z - 聚焦到选中物体
└── F3 - 重置物体位置（仿真模式）
```

### 视图模式
```
渲染模式：
├── 标准渲染 - 真实感渲染
├── 线框模式 - 显示网格结构
├── 材质预览 - 材质效果展示
└── 物理调试 - 显示物理信息
```

## ✏️ 编辑栏

### Transform属性编辑
```
Position (位置)：
├── X轴位置 - 世界坐标X方向
├── Y轴位置 - 世界坐标Y方向  
└── Z轴位置 - 世界坐标Z方向

Rotation (旋转)：
├── X轴旋转 - 绕X轴旋转角度
├── Y轴旋转 - 绕Y轴旋转角度
└── Z轴旋转 - 绕Z轴旋转角度

Uniform Scale (统一缩放)：
└── 缩放比例 - 等比例缩放系数
```



### 编辑操作方式
```
数值输入：
├── 直接点击输入框输入精确数值
├── 使用键盘上下箭头微调
└── 按Enter确认修改

滑动调节：
├── 鼠标悬停显示滑动箭头
├── 拖拽箭头实时调节数值
└── 实时预览变化效果

批量编辑：
├── 多选物体同时编辑
├── 相对变化应用
└── 重置到默认值
```

## 📦 资产与终端栏

### 功能模式切换
```
资产模式：
└── 管理和使用已订阅的资产

终端模式：  
└── 显示仿真程序运行信息
```

### 📋 资产模式功能
```
资产管理：
├── 展示所有已订阅资产
├── 支持模糊搜索过滤
├── 显示资产详细信息
├── 拖拽资产到场景
└── 打开资产库入口

操作功能：
├── 搜索框 - 快速过滤资产
├── 资产预览 - 悬停显示信息
├── 拖拽添加 - 拖拽到视图使用  
├── 信息复制 - 复制资产信息
└── 打开资产库 - 跳转到资产库网站
```



### 💻 终端模式功能
```
信息显示：
├── 仿真程序启动信息
├── 运行过程日志输出
├── 错误和警告信息
├── 性能统计数据
└── 程序退出状态

管理功能：
├── 清空终端输出
├── 复制终端内容  
├── 搜索日志信息
└── 导出日志文件
```



## 🎨 界面个性化

### 布局调整
```
支持功能：
├── 拖拽调整面板大小
├── 隐藏/显示特定面板
├── 保存自定义布局
└── 恢复默认布局

快捷操作：
├── 双击面板标题 - 最大化面板
├── 右键面板 - 显示面板菜单
└── 拖拽面板边界 - 调整大小
```

### 主题设置
```
视觉选项：
├── 深色主题/浅色主题
├── 字体大小调节
├── 界面缩放比例
└── 颜色方案自定义
```

## 🔄 工作流集成

### 典型使用流程
```
1. 菜单栏 → 创建或打开项目
2. 资产栏 → 搜索和添加资产  
3. 视图栏 → 构建和编辑场景
4. 大纲栏 → 管理场景层级
5. 编辑栏 → 精确调整属性
6. 菜单栏 → 运行仿真程序
7. 终端栏 → 监控运行状态
8. 菜单栏 → 保存项目结果
```

### 效率提升技巧
```
快捷键使用：
├── Ctrl+S - 快速保存布局
├── 1/2/3 - 切换编辑工具
├── Delete - 删除选中物体
└── F3 - 重置仿真状态

多面板协作：
├── 大纲选择 + 编辑栏调整
├── 资产栏搜索 + 视图栏添加
├── 视图操作 + 终端监控
└── 相机切换 + 场景编辑
```

OrcaLab的模块化界面设计充分考虑了仿真开发的工作流程，各模块功能明确分工，同时保持良好的交互协作，为用户提供高效的开发环境。
