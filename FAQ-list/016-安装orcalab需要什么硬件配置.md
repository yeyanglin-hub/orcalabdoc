# 安装OrcaLab需要什么硬件配置？

## 问题
运行OrcaLab需要什么样的硬件配置？最低配置和推荐配置分别是什么？

## 回答

OrcaLab作为高保真物理AI仿真系统，对硬件配置有一定要求，特别是显卡性能。以下是详细的硬件配置要求：

## 🎮 显卡要求（核心重要）

### ✅ 推荐显卡配置
```
NVIDIA RTX 40系列：
├── RTX 4090 (推荐)    - 顶级性能，大型场景流畅
├── RTX 4080 (推荐)    - 高性能，复杂仿真适用  
├── RTX 4070 (良好)    - 中高性能，一般使用流畅
└── RTX 4060 (基础)    - 入门级，基础仿真可用

NVIDIA RTX 50系列：
├── RTX 5090 (最佳)    - 最新顶级性能
└── RTX 5080 (优秀)    - 最新高端性能
```

### 📋 驱动版本要求
```bash
RTX 40系列：驱动版本 ≥ 535.00
RTX 50系列：驱动版本 ≥ 550.00

# 检查当前驱动版本
nvidia-smi
```

### ⚠️ 不支持的显卡
- ❌ 集成显卡（Intel UHD, AMD APU）
- ❌ AMD独立显卡
- ❌ NVIDIA RTX 30系列及以下
- ❌ NVIDIA GTX系列

## 💻 CPU配置要求

### 推荐配置
```
Intel处理器：
├── Intel i9-13900K/14900K (最佳)
├── Intel i7-13700K/14700K (推荐)  
└── Intel i5-13600K/14600K (良好)

AMD处理器：
├── AMD Ryzen 9 7900X/7950X (最佳)
├── AMD Ryzen 7 7700X/7800X3D (推荐)
└── AMD Ryzen 5 7600X (良好)
```

### 最低配置
```
Intel i5-12400 或 AMD Ryzen 5 5600X
6核心12线程以上
基础频率 ≥ 3.0GHz
```

### 性能对比
| CPU等级 | 仿真复杂度 | 多任务能力 | 推荐场景 |
|--------|-----------|-----------|----------|
| 高端 (i9/R9) | 复杂大型场景 | 强 | 研究级应用 |
| 中端 (i7/R7) | 中等复杂场景 | 较强 | 教学与学习 |
| 入门 (i5/R5) | 基础简单场景 | 一般 | 入门体验 |

## 🧠 内存配置要求

### 推荐配置
```
32GB DDR4-3200 或更高 (推荐)
- 支持大型复杂场景
- 多个仿真并行运行
- 大量资产加载
```

### 最低配置
```
16GB DDR4-3200 (标准)
- 中等规模场景
- 单个仿真流畅运行
- 基础资产管理
```


```
8GB DDR4-2666 (最低)
- 小型简单场景
- 基础功能体验
- 可能出现性能瓶颈
```



## 💾 存储配置要求

### 推荐配置
```
SSD存储方案：
├── 系统盘：500GB+ NVMe SSD
├── 资产盘：1TB+ SATA SSD  
└── 数据盘：500GB+ SSD (可选)

总存储：2TB+
```

### 最低配置
```
混合存储方案：
├── 系统盘：250GB SSD (必需)
├── 资产盘：500GB HDD (可用)
└── 总存储：750GB+
```

### 存储用途分析
```
空间分配建议：
├── 系统与软件：100GB
├── OrcaLab资产库：200-500GB
├── 仿真数据：100-300GB
├── 用户项目：100-200GB
└── 系统预留：100GB
```

## 🌐 网络要求

### 带宽要求
```
推荐带宽：100Mbps+ (下载)
最低带宽：50Mbps (下载)

用途：
- 资产包下载
- 软件更新
- 云端服务连接
```

### 网络稳定性
- 低延迟连接（<50ms到资产库）
- 稳定连接（避免频繁断网）
- 无严格防火墙限制

## 📱 外设要求

### VR设备（可选）
```
支持的VR设备：
└── Pico Ultra 4 + 配套手柄

用途：VR遥操作与数据采集
```

### 输入设备
```
必需：
├── 键盘 (支持标准快捷键)
└── 鼠标 (三键滚轮鼠标推荐)

推荐：
├── 机械键盘 (提升操作体验)
└── 游戏鼠标 (精确控制)
```

## 🖥️ 显示器要求

### 推荐配置
```
分辨率：2560×1440 (2K) 或更高
尺寸：27英寸以上
刷新率：60Hz以上
色彩：sRGB 99%+ (设计需求)
```

### 最低配置
```
分辨率：1920×1080 (1080p)
尺寸：24英寸
刷新率：60Hz
```

### 多显示器支持
- 支持多显示器扩展
- 推荐双显示器提升工作效率
- 主显示器用于3D视图，副显示器用于工具面板



## ⚡ 性能优化建议

### 系统优化
```bash
# 关闭不必要的后台程序
# 设置高性能电源模式  
# 定期清理系统垃圾
# 保持驱动更新
```

### BIOS设置
```
启用：
├── XMP/DOCP (内存超频)
├── PCIe 4.0 (支持的话)
└── 虚拟化技术

禁用：
├── 集成显卡 (如有独显)
└── 不必要的板载设备
```

## 🔍 硬件检查命令

```bash
# CPU信息
lscpu | grep "Model name"
cat /proc/cpuinfo | grep "cpu cores"

# 内存信息
free -h
dmidecode -t memory | grep -E "Size|Speed"

# 显卡信息  
lspci | grep VGA
nvidia-smi

# 硬盘信息
lsblk
df -h

# 网络速度测试
speedtest-cli
```

选择合适的硬件配置是确保OrcaLab流畅运行的基础。建议根据使用需求和预算选择合适的配置方案。
