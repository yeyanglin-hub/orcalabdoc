# VR数据采集需要什么设备？

## 问题
OrcaLab支持VR遥操作进行数据采集。为了实现这一功能，我需要准备哪些VR硬件设备和相关软件环境？

## 回答

OrcaLab的VR遥操作数据采集功能旨在通过沉浸式体验，高效地生成高质量的机器人示教数据。要实现这一功能，您需要准备特定的VR硬件设备和相应的软件环境。

## 📋 VR数据采集的设备要求

### 1. **VR头显 (VR Headset)**

#### 推荐设备
-   **Pico Ultra 4**：OrcaLab官方文档中明确提及并支持的VR设备。确保其及其配套手柄功能正常。

#### 替代/兼容设备 (可能需要额外配置)
-   其他支持PC VR模式的VR头显，可能需要进行额外的配置或驱动安装。具体兼容性请参考OrcaLab官方文档。

### 2. **VR手柄 (VR Controllers)**

-   **配套手柄**：VR头显通常会配备一对交互手柄，用于在虚拟环境中进行操作。
-   **功能映射**：OrcaLab的VR遥操作功能会将手柄的按键和姿态映射到机器人控制上（如机械臂关节控制、抓取开关）。

### 3. **高性能PC主机**

VR遥操作和OrcaLab仿真都需要强大的计算能力。PC主机需要满足以下要求：
-   **GPU**：NVIDIA RTX 40/50系列显卡（具体要求参见[安装OrcaLab需要什么硬件配置？](../FAQ/016-安装orcalab需要什么硬件配置.md)）。
-   **CPU**：高性能多核处理器。
-   **内存**：16GB或以上RAM。
-   **硬盘**：SSD硬盘，确保加载速度。
-   **操作系统**：Ubuntu 22.04 LTS 或 24.04 LTS。

### 4. **USB连接线**

-   **高质量USB数据线**：用于连接Pico VR头显与PC主机，确保稳定的数据传输和调试连接（ADB连接）。

## 📋 VR数据采集的软件环境要求

### 1. **OrcaLab客户端**
-   已成功安装并能正常运行的OrcaLab客户端。通常以"无仿真程序（手动启动）"模式启动，作为后端仿真器。

### 2. **OrcaGym OrcaManipulation代码库**
-   这是一个开源代码库，包含VR遥操作和数据采集相关的Python脚本。您需要从GitHub获取：
    ```bash
    git clone https://github.com/openverse-orca/OrcaManipulation.git
    cd OrcaManipulation
    conda activate orcalab # 激活OrcaLab的Conda环境
    pip install -r requirements.txt # 安装项目依赖
    ```

### 3. **PicoController.apk**
-   这是Pico VR头显上需要安装的配套控制程序（或称"OrcaGymCtrl"）。
-   **路径**：通常位于`OrcaManipulation/src/examples/超市场景青龙机器人数采案例/pico安装包/`目录下。
-   **安装方式**：
    -   将`PicoController.apk`复制到PICO设备目录中。
    -   在VR设备中找到文件管理器，使用手柄点击A键确认安装。



### 4. **Android调试工具 (ADB)**
-   用于建立PC与Pico VR头显之间的通信连接。
-   **安装**：在Ubuntu终端中执行：
    ```bash
    sudo apt install android-tools-adb android-tools-fastboot
    ```
-   **连接命令**：
    ```bash
    adb reverse tcp:8001 tcp:8001
    ```
    -   **注意**：每次重启PC或断开PICO的USB连接后，需重新执行此命令。

### 5. **VR设备开发者模式**
-   Pico VR头显需要开启"开发者模式"才能进行USB调试和`adb reverse`连接。
-   **开启方式**：在VR视角中，点击"设置" -> "关于本机" -> "软件版本号"，连续点击6-7次调出开发者模式，然后打开"调试开关"。



### 6. **场景资产包**
-   您需要订阅与VR数据采集任务相关的3D场景和机器人资产包（如`ShopScene_Scaning`和`openloong`），并在OrcaLab客户端中加载它们。

## 📝 总结

VR遥操作数据采集需要一套完整的硬件和软件环境。硬件上包括**Pico Ultra 4 VR头显及手柄**、**高性能PC主机**和**USB数据线**。软件上则需要**OrcaLab客户端**、**OrcaManipulation代码库**、**PicoController.apk**、**ADB工具**以及**开启VR设备开发者模式**。确保所有这些组件都正确准备和配置，才能顺利进行VR数据采集。
